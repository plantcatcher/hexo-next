我将开启一个新的习惯

发表于 2026-04-07 分类于 创业

我迷茫的时间，一般不过半个月。在低迷一段时间后，我会不自觉地开始寻找解决方法，去掉迷雾。这篇博文将是我养成一个新习惯的开端：整理一天的观影、阅读的思考，并记录下来。——有点像高中时期写的读后感，用读后感的名义，实则写对生活、对社会、对自己、对人生的看法。

阅读也好，看纪录片也好，我的最终目的是促进思考、启迪思想。看后不经过思考的内容，顶多算娱乐（刷短视频、看八卦新闻等），而我目前阶段，思考以及促进思考的时间要占到我日常生活的80%，也就是说扣除休息的8个小时，剩下的16个小时中，13个小时用在学习、阅读、思考上，真正娱乐的时间只能有3个小时（玩商业游戏算在里面吗？不一定）。

今天中午醒来，到天台阅读《咸的玩笑》最后一点内容。*在这里反思一下之前的、错误的计划，即整理一个已读书单，然后每看完一本书就写博客记录下来——太过刻意、笨拙，实际操作并不经过思考，仅仅是搬运工。我还是习惯有感而发，习惯不强迫。*回到《咸的玩笑》这本书，我总共花了3天把这本书看完，这算是第一本我真正意义上看完的刘震云的小说。这本书确实给我很多启发，但在我目前阶段来说依旧算是偏放松性质的——商业传记、金融科普、经济原理是我当前的主轴。因此这本书不想做太多思考总结，阅读过程中有感悟就行。

这本书结束后，我立马打开了新的书《哈萨比斯：谷歌AI之脑》，我阅读这本书的目的是——我失去了目标，厌烦了各种商业、经济、金融的书籍，需要一本创业的传记来给自己打打鸡血。当天看了三四个小时吧，看下来，这本书很值得接着阅读。

差点忘了一件事，我昨天花了4个多小时，玩完《芯片大亨》这个游戏，虽然也写了一篇博文，但总归是太模式化，现在完全可以再多写一些。总而言之，言而总之，这个游戏的商业模式还是较为简单的，也能让我多了解一些关于芯片相关的知识，总归是好的。然后我今天又在taptap上找了一些和商业金融相关的游戏，打算都玩玩——当作阅读、学习之余的放松之选。（之前习惯在去参加活动的地铁、公交上，看pitch商业文章、看知乎新闻，现在完全可以用玩游戏取代）

晚上，我浪费了些时间，想在云服务器上安装open claw，弄了一阵子没弄出来，担心会损坏之前部署的机场功能，所以放弃。AI训练营的内容也就没怎么看完，明天再看吧。然后就是转到了Qclaw上，我发现这个工具挺好的，所以之后几天先折腾吧，毕竟每天4000w的token不用白不用。

再之后，看了两篇挺有价值的长视频。一个是关于诺贝尔经济学奖的，一个是货币的纪录片第一集。经济学奖的内容，首先探讨了马尔萨斯陷阱，大致就是人口曲线和食物曲线的相交，是较为原始的解读。中间出现了一个经济学家，证明了马尔萨斯陷阱是错误的，说技术进步可以突破这个陷阱。然后这次经济学奖颁发的就是进一步说明为什么技术进步能够突破这个陷阱。（我写的过程中，发现自己完全没理解透这个视频，在思考要不要重新看一遍）。纪录片《货币》第一集，我又记住多少呢？老实说并没有多少，但是至少让我意识到：货币交易其实本质很简单，就是价值交易。其它的各种商业模式、企业模式、产业集群都只是其中的衍生品，我并不需要都了解，了解对我也没太多的帮助。例如，体育产业、房地产、期货市场，刚开始都是很简单的一个点，然后慢慢衍生出类似二级市场、相关服务市场等内容，我当时很好奇，为什么有些书推荐我看微观经济学，而非宏观经济学，现在忽然意识到宏观经济学是从微观经济学出来的，微观经济学是最初的那个“点”，能让我更好的理解经济、理解金融、理解商业。

最后，再说说我的计划：**每天观看一集纪录片，并写博客记录我对当前学习内容的思考和感悟。**目前暂定的纪录片是《货币》，央视出品，豆瓣评分一般，但是依旧值得看。

玩《芯片大亨》手游的碎碎念

发表于 2026-04-06 更新于 2026-04-08 分类于 创业

一开始，是想玩电脑端的商业游戏，来提升个人财商。但稍显麻烦，所以想到TapTap上有很多游戏，因此开始玩商业手游。《芯片大亨》这款手游算是第一个我正经玩、并且通关了的游戏，花了4个小时熬夜通关了。

那么我从这款游戏中得到了什么启发或者感悟呢？

首先，就是各种基础的芯片知识。例如DUV光刻机、EUV光刻机、HEUV光刻机，反正就是越来越厉害。以及制作芯片需要的材料，包括硅片、光刻胶、化学品等，接单和生产的时候需要考虑库存。接小公司的单子，赚不了多少钱；承接需要流片的项目，能够获得更高的利润，并且一单就能吃很长时间。

至于商业方面，就是接商单——研发工艺——提升良率——扩大产能（工厂+设备）——赚钱。最终就是越做越大，何止财务自由，简直就是螺旋升天。但就个人而言，卡在了接商单上。我没有商业点子，因此根本没到通过宣传来接订单这一步。但我知道我之后选择的商业点子，必须是能扩大规模，且最终能成为一个系统，让我当甩手掌柜的那种。

**总结：4个小时，刷手机很快结束，阅读差不多也就一本书1/3的内容，所以尚可接受。**我依旧认为，接触更多的信息（非阅读这种单一信息源），能更好的将各种知识融会贯通，对我的帮助会更大。所以，多试试各种方法准没错儿。

2026-4-8 更新内容：

现实情况下，你不可能从事多份不同行业的工作，然后学习行业内的知识，但是可以通过玩游戏模拟器来达到同样的效果。然后我和豆包聊了一会儿，发现这个想法是对的，但他给我的一个建议是，玩完一个游戏了之后，可以查找那个行业相关的真正的知识，然后了解一下真正的商业运作。因此我现在玩完《芯片大亨》之后，打算了解一下真实世界中芯片制造的SOP和相关信息。

让AI以讲故事的方法，将整个行业的流程与注意细节告诉我，然后我遇到不懂的点就通过与AI对话来理清楚，最后我对整个行业进行自己理解的话来总结，让AI判断我理解的对不对。最后AI将对话、知识点归纳、总结。

一、芯片行业核心角色分工（最易混淆点）

1. 纯芯片设计公司（Fabless，只设计不生产）

  • 高端设计：高通（骁龙）、联发科（天玑）、苹果（A/M 系列）、英伟达、AMD、华为海思，主打 CPU、GPU、手机主芯片 SoC，设计难度极高，是行业暴利环节。
  • 小众设计：手机影像 ISP 芯片、音频芯片、快充芯片、WiFi 蓝牙芯片，功能单一、难度低，国内大量厂商可做，利润也很可观。
  • 核心工作：只做芯片图纸设计，无工厂、无光刻机，设计完交给代工厂生产。

2. 晶圆代工厂（Foundry，只生产不设计）

  • 头部代表：台积电（全球第一，专攻先进制程）、中芯国际、华虹（国内主力）。
  • 核心工作：承接设计公司的订单，用光刻机等设备完成芯片流片、制造，只赚代工费。
  • 我玩的《芯片大亨》，核心身份就是台积电这类晶圆代工厂，主打接单、扩产、提良率。

3. IDM 公司（自己设计 + 自己生产，全链条通吃）

  • 代表企业：三星、SK 海力士、美光、长鑫、长江存储。
  • 核心模式：自己完成芯片设计，自建工厂、自购光刻机，全程自主生产，不依赖第三方代工厂。
  • 特殊点：三星是全球唯一全产业链覆盖，既做逻辑芯片设计 / 代工，也做存储芯片，还做封测、终端销售。

4. 封测厂（产业链末端，薄利环节）

  • 代表：日月光、长电科技、通富微电。
  • 核心工作：代工厂造出晶圆后，负责切割、封装、测试，做成成品芯片，大多交给第三方，台积电极少自己做。

二、芯片分类及核心区别

1. 逻辑芯片（负责运算处理）

  • 包含：CPU、GPU、手机主芯片、影像芯片、音频芯片、快充芯片、WiFi 芯片。
  • 特点：功能多样，高端款迭代快（手机主芯片每年更新），小众款设计简单，全部找台积电 / 中芯等代工厂代工。

2. 存储芯片（负责数据存储，迭代慢）

  • 运行内存（DRAM）：手机 / 电脑运行内存，断电数据丢失，速度快，代表：DDR5、HBM。
  • 闪存存储（NAND）：手机机身存储、U 盘、SSD，断电不丢数据，代表：UFS、固态硬盘。
  • 特点：设计稳定、迭代周期长（3-5 年换代），一套成熟设计可长期大规模量产，均由 IDM 厂商自产，不找台积电代工。

三、光刻机与国内现状

1. 光刻机类型与禁售情况

  • EUV 光刻机：最先进，用于 7nm 及以下芯片，荷兰 ASML 垄断，对国内完全禁售，国内 0 台。
  • DUV 光刻机：中高端制程主力，分浸没式 / 干式，高端 DUV 严限制售，老式 DUV 可正常购买，国内芯片厂均为 DUV + 老式光刻机。

2. 国内光刻机制造

  • 唯一量产芯片制造光刻机：上海微电子（SSA600/90nm、SSA800/28nm），可满足成熟制程需求。
  • 国内总量：约 700-800 台，全为 DUV 及老式机型，无 EUV；高端 DUV 靠存量，新购受限。

3. 逆向拆解误区

光刻机可以拆开研究、借鉴思路，但无法靠拆解复制，核心是材料、精度、软件算法、工业体系差距，不是单纯图纸问题。

四、芯片生产全流程

  1. 芯片设计公司完成图纸设计→2. 交给晶圆代工厂（台积电 / 中芯），用光刻机制造晶圆→3. 第三方封测厂封装测试→4. 返还给设计公司→5. 设计公司加装散热、组装后对外销售。

五、各环节利润率（从高到低）

  1. 高端芯片设计（英伟达、高通）：毛利 55%-70%，暴利
  2. 台积电（先进制程代工）：毛利 56%-60%，高利润
  3. 国内小众芯片设计（影像 / 快充）：毛利 35%-45%，中高利润
  4. ASML（光刻机厂商）：毛利 52%+，设备 + 售后双赚钱
  5. 中芯 / 华虹（成熟制程代工）：毛利 18%-25%，薄利
  6. 封测厂：毛利 20%-30%，净利最低，量大利薄

六、ASML 盈利模式（不止卖设备）

  1. 设备销售（占 75%）：EUV/DUV 光刻机一次性售卖，EUV 单价超 20 亿人民币，全球垄断。
  2. 售后与软件（占 25%）：长期维保、备件更换、设备升级、软件订阅收费，光刻机寿命 10-20 年，靠售后长期稳赚，续约率超 90%。

七、关键补充常识

  1. 存储芯片（内存 / 闪存）也需要光刻机，生产设备、设计软件和逻辑芯片基本一致，只是内部电路结构不同。
  2. 国内影像、音频、快充芯片，无需高端制程和台积电，中芯、华虹等国内普通代工厂就能生产，完全绕开高端限制。
  3. 先进制程芯片（7nm/3nm）必须靠高端 DUV/EUV，成熟制程（28nm 及以上）国内可自主实现。

关于失去目标与无意义坚持的思考

发表于 2026-04-06 分类于 创业

1 我当前的困顿

我曾尝试将博客经营得井井有条，但发现自己在坚持过程中慢慢变得不耐烦。正常，如果我真能保持下去，就不是INFP而是INFJ了。我更喜欢的记录方式，还是这种想到哪是哪儿，不需要规划的样子。

说起来，我为什么失去了目标呢？我最近参加的活动，是在进行TEDxShenzhen面试后，参加一人公司跨境电商的活动，然后还去了跨境电商的展会。我不知道问题出在哪里，其实春节从老家回到深圳，我就已经没了当初的动力了。

我依旧坚持阅读：《美丽与悲哀》《奥尔特曼传》《黄仁勋：英伟达之芯》《小狗钱钱》《消失的13级台阶》《百万富翁快车道》《咸的玩笑》，这些全是3、4月份读的书。

我仍坚持锻炼财商：玩CashFlow现金流游戏（隔三岔五玩），玩手游芯片大亨（通宵4个消失通关），看Pitch的商业文章，看商业大佬的访谈…但是我依旧不知道自己的出路在哪儿。我知道商业的整体框架，也知道应该对各个节点逐个击破，但我似乎丧失了动力。

金钱的缺失，甚至于手机流量的缺失，让我不自觉将注意力放到了书籍和各种所谓“自我提升”的行为上。但也让我迷失在“何为生活”的困顿中。一个月前我曾志得意满的认为，当下的努力肯定会对未来的我有帮助。TED的面试甚至还加强了自我认可，让我觉得可以申请深圳科创学院了。那么，问题出在哪儿呢？

2 我所能做的改变

我需要知道问题出在哪儿吗？似乎也不需要。

面对问题，解决问题。遇到困难，解决困难。我需要做的是做出改变，去解决当前的困顿。该做什么，这才是我需要考虑的。

做什么呢？下面是我能做出的改变：

  1. 放弃过于规范化的写博客方法，采用更随意的想到什么写什么；
  2. 坚持每天写博客，一天至少写一篇；
  3. 坚持每天阅读3小时以上，但没必要每本书都写读后感；
  4. 增加每天玩商业游戏1小时，可以是电脑游戏，也可以是手游；
  5. 要不要每周去一趟图书馆呢？还不确定；
  6. 要不要每周参加一次活动呢？还不确定；
  7. 要不要每周去街拍呢？还不确定；
  8. 要不要找女朋友呢？还不确定；

3 再哔哔一会儿

有一次我从后海的活动结束回出租屋，在坐地铁的途中，一个在昆士兰大学留学的朋友给我打电话，由于信号不行所以我中途出地铁在大厅打电话，一不小心打了2个多小时。地铁扣超时费15元，我发到小红书然后很多人评论。其中有人羡慕还能找到打电话这么长时间的朋友——原来我曾习以为常的事情，这么多人是缺少的。忽然间，我很感恩自己能有这样的经历。我并不是自己认为的这样一无是处，一事无成。每个人都有价值。

另外， 我是实打实认识到，通过坚持写文章输出，能够整理思绪，还能为接下来的行动制定方案和做心理准备。好处很多，坚持下去吧。

关于地理自媒体内容再创作

发表于 2026-03-22 分类于 创业

我是一个原创地理科普内容创作者，地图、文章、视频都是以原创为主。但我经历过很多次，我发布的内容浏览量平平无奇，而别人盗用我的原创地图，加个bgm和一些文字，竟然能爆火，以至于让我刷到。

这让我意识到内容质量固然重要，但呈现方式更需要注意。所以我计划将过去制作的地图视频化。我会依旧更新最新的原创地图，但确实可以用“新瓶装旧酒”。

思考之后该做什么

发表于 2026-07-21 分类于 创业

这段时间，终于是从番茄小说创作的热情中出来了，然后进入对公众号内容搬运上，刚对搬运到知乎上头，想不到检测出AI然后禁言7天。算了，只搬运到博客吧，我的博客目前每天观看人数挺多的其实。但我隐约觉察到博客更多适合攻略、教程类文章，像地理科普的点击率其实不高。另外，我将地图作品上传博客了，应该也能带来另一波流量。

创业其实是暂缓了的，但我还在时不时思考怎么赚钱。输入其实也是暂停了的。有时候我都不知道一天天是怎么过去的，啥也没做就消失了。我打算打零工了，先把每个月的生活费赚到手。

我这个人，确实如《稀缺》中所呈现的状态，当面临资金压力时，各方面的表现都挺差的。没能提升自己，反倒焦虑开始病急乱投医了。赚到钱又可能盲目自信，总之问题很多。

说实话，我很不认可自己现在的生活，我鄙夷、我不喜欢。但我要做什么来改变这些事情呢？不知道，我先去倒个垃圾。

我的脑子，每天想的事情很局限，我能感受出来。毫无乐趣。睁眼闭眼都是这些“无解”的事情，生活有什么意思呢。要如何破局？Emmm……要如何破局？不能靠阅读、不能靠观影、不能靠沉浸于创作、不能靠参加活动、不能靠刷手机、不能靠聊天…那么该靠什么呢？靠自己？

开始瞎逼逼。在成都时候，我每天都去成都天府新区的图书馆“假装学习”，由于没有反馈机制和评审机制，所以完全不知道自己学得怎么样。我当时的学习模式，依旧是那种跟随感觉，胡吃海塞，期待某天能顿悟的那类。这种事情，放到现在依旧有教训意义，也就是说假使我今天开始每天去图书馆学习，但是学什么呢？学到什么程度呢？完全就是一头雾水。想靠着行为本身来缓解焦虑，很无意义。

我要找到焦虑的源头。我想，可能是需要找到一种切实可行的盈利模式，然后能够实现收支平衡，我的现状以及独立思考能力才有可能恢复。不然，我只能是先去打零工养活自己。因为整体无所事事完全无济于事。

你把你的未来交给谁？书籍、活动、人脉，还是自己？

发表于 2026-07-19 分类于 创业

参加完第一届AI恐怖片黑客松，感慨现在大家真有才，真有活力，真青春。回到出租屋，将地理科普文章copy到知乎、博客后，我就躺在床上刷手机、短视频了——都是一些不进脑子的东西。 然后听播客准备睡觉，忽然“垂死病中惊坐起”，意识到一个点：我应该将自己的未来交给谁？

不废话的说，当然是交给自己了。但，我回顾自己这段时间，发现自己迷茫时，却寄希望于大量的阅读、大量的信息输入、频繁的参加活动——我当时期望的是什么？是希望量变引发质变，是灵光乍现，是顿悟。但说实在的，太虚无缥缈了。

真正对自己负责，是将时间花在思考上。不一定看的越多，知道的越多，就能更好的解决问题。解决问题的能力，是不靠那些的。哪怕不论解决问题，我当下的生活的转变，靠的不是那些“重要不紧急”的日常积累，而是“重要且紧急”的执行力。遇到问题，解决问题；发现不足、弥补不足；了解优势，发挥优势。这才是正确的方式。

所以，与其花时间去大量输入可能对现在毫无帮助的信息，不如让自己头脑冷静下来，踏踏实实思考下一步怎么走。

要勤动脑筋，要1天做3天的活儿

发表于 2026-07-14 分类于 创业

如果我真要安排自己的生活，其实可以安排得很满，因为我有的是想法去实现。输入部分，阅读、英语博客访谈等可以占我每天生活的3小时；输出部分，番茄小说撰写、公众号日更、视频制作尝试、网站or小程序产品开发，以及思考和博客记录，这些能占6-8个小时。

玩金钱游戏，那么我需要了解规则，熟悉评分体系，然后得分，最终夺冠。

准备开始进行视频制作？

发表于 2026-07-12 分类于 创业

当我的地理科普公众号重新进入正轨，我就可以将精力放到另一个方向：创业公众号or地理科普视频制作。我跟ChatGPT探讨过，关于输入可以通过交叉学习，但是输出最好还是能集中一整天进行产出，中间不要换另一种产出内容。至于需要发散性的、灵感型的内容创作，例如微电影，则另说。

昨天又和小伙伴微信通话聊了五个多小时，有点启发，但是不想在这里多说。我为什么突然在凌晨想写这篇博客，是因为在刷YouTube时，刷到一个YouTuber一共有4W粉丝，每个月的播放量在30-70w之间，每个月收入在1000-3000美金。我算下来差不多一天需要1w的播放量，每天收入200+人民币，也就是说ecpm在20元左右。和我的公众号区区2元相比，那可是翻了十倍。还有就是，她的视频剪辑很简单，并没有花里胡哨的动效，但依旧能吸引别人关注，我想我也可以不用太关注、太去琢磨把视频做得完美，只要及格就可以发布到YouTube上了。回想小红书和公众号，一开始的内容也是惨不忍睹，也是在不断尝试中，找到最好的表现方式。重点还是，先做先发。

然后我还刷到了一个国外女性做地理科普视频的，这一期视频是关于南极科考的。我也想未来能通过自媒体实现环游世界的梦想，所以我立马从床上起来开始写这篇博客。

说起创业公众号，我打算花两天时间，将之后半个月的文章素材准备好，并定时发布。之后的精力就全放在地理科普视频的制作上。我对创业公众号并不太抱有期望，因为我的唯一实名认证机会已经给了【那山那海那座城】的公众号了，这个创业公众号哪怕做起来了也不能开通盈利。但为什么还要做这个号，因为我需要将学到的东西写出来，或者说将思绪整理清楚 ，把问题显像化。

身边创业的小伙伴都在前进，快速的前进，而我还在慢吞吞的爬着。自由职业者最需要克服的，其实是对自己生活的掌控，对自己行为的管控。我需要想一些法子让自己加加速了。

从需求出发，去分析市场与产品开发

发表于 2026-07-08 分类于 创业

需求不会因为科技发展而发生剧烈变化。具体而言，虽然AI Agent出现后，网站、软件可能不再是主流场域，但是它们所解决的需求和痛点还是在的，只不过是转移了。因此，站在需求和痛点的视角出发，其实并不在乎产品是以什么样的形态出现，只要能解决问题就行。况且，在我分析下，短期内网站、软件还将会占据很大的市场，结合vibe coding这个技术平权的工具，我完全有机会大杀四方。

刚才，一个创业的小伙伴Aris给我发来她开发的一个网站，是针对AI创业者的兼职平台，能够发布需求or接单。我联想到，当用户增加可能会发布很多有意思的内容，像一个大型的人类观察室。我的脑子忽然开始思考，把握人性，我能做很多有意思的产品。

恰好我的公众号又步入正轨了，每天都有50块的收入。因此完全可以尝试更多可能性，无论是发布各种视频测试市场、开发各种有意思的产品去吸引用户，只要我去掉浪费时间的各种行为，专注在学习、阅读、输出、开发上，总能在深圳创出一片天地的。

看雷军的创业课程

发表于 2026-07-06 分类于 创业

在看小米的创业课程【雷军创业课：保姆级创业教程，创业者必看！】，很有意思。讲了创业要go to big market，还要给公司起一个好名字。关于融资，你需要在创业前就给身边的朋友留下信任，证明你能做成事儿。

商业计划书BP没多大用，一个靠谱人的推荐和信用，在整个投资领域里面是决定性的东西。

关于商业模式这一块，等我真正创业成功，需要跟投资者接触的时候，可以再翻出来看看。

什么时候启动融资？当钱花到一半的时候。因为此时还有谈判筹码，不至于不融资明天就饿死。

其实创业者，唯一唯一的资产是什么？就是那百分之百的股份而已。

我发现，当没有心思在阅读商业书籍时候，随机刷到的大佬公开课还是很有收获的。因为现在自媒体的声音很杂，无论是有点名气的创业者、投资人，甚至是传声筒，都在宣扬自己的理念，“传授”自己的经验。与其研究这些质量参差不齐的内容，不如学习自己认可的商业大佬的理念，而且这些大佬的理念够我学很长时间——无论是大佬的人数，还是大佬的经验。

另外，确实最好的在学习完后，在博客上记录自己的感悟、思考。虽然我知道以后肯定不会回看，但是还是值得记录。

从高考查分的视频集合看国内应试教育

发表于 2026-06-27 更新于 2026-07-03 分类于 创业

在B站开始刷到高考查分的人生百态，我是带着焦虑且拧巴的心态坚持看下去的。加上看到一些专科和985的对比，让我对国内应试教育体制又开始进行思考了。

我不想论应试教育对错与否，我认为不同的社会、环境需要不同的筛选的标准。我只是想就我个人而言，我需要当下的许多精力、自我安慰去弥补过去教育对我心理造成的“创伤”与应激反应。哪怕我现在生活并不坏，但是那些关于教育的不和谐、难堪的一幕幕还是会影响当下的自我判断，且久久无法释怀。

实话说，我的青春时代并不值得回忆。不幸的人用一生去治愈童年（青春）。青春伤痛文学？不，我的记忆碎片套不上任何一部青春文学。随着年龄的增长，我以为我能放下那些不堪的记忆，但每当出现关于judge，关于等级、关于集体的类似情况出现，我总会在某一刻下意识用那个敏感、脆弱、无助、迷茫的青少年的视角看着周遭。——PS:这是不是能写出一个挺好的剧本？

在社会中，我渐渐发现存在别的评价体系，它并不以成绩论成败，也不以集体主义为荣。而我目前处在一种混乱的评价体系之中，摸不清规则，找不到合适的努力方向。我的性格，是小镇做题家，有很强的冲劲，甚至可以说是野心。我横冲直撞，尝试了很多种不同的生活方式，但并没有找到稳定且自在的状态。我很羡慕那些打小内核稳定的人，我很有幸遇到几个，而且他们很早就脱离了常规的应试教育的体系。Nina、Jennie、James…我相信他们未来肯定成就斐然。

待会儿我会跟昆士兰大学的一个小伙伴通电话，让她帮我梳理一下思绪。

我常常会通过大量阅读、观影、四处旅行、参加各种活动，采用量变产生质变的想法去执行。说到底，没有任何章法而言，却意外有效，能唬住挺多人的。我的自我认知，却将自己称作“纸老虎”，没有多少能力。当“紧急不重要”或者“紧急且重要”的事情出现，我本能会逃避、拖延、自我贬低，那种不配德感会出现，非常影响正常生活。怎么解决呢？想不明白。

聊聊近况，准备参加深科创的面试

发表于 2026-06-26 分类于 创业

本来想着昨晚写这篇博客，但是推到了今晚。先说说让我下决心动手的原因：在YouTube看【脑洞乌托邦】关于段永平和孙宇晨（期间提到李笑来），他们的故事让我意识到将自己的想法记录下来的重要性。

一个人，要怎么才能在商业金融领域脱颖而出？是资金还是技术？都不是，我认为是经历。这就像是战争中打仗一样，你的功勋比身出武门或者纸上谈兵更有价值。所以就我个人而言，AI技术的提升对我的帮助有限，我能在空闲时刻去玩玩，但是精力还是得放在别的地方。那么我更多的精力和时间该放到哪里呢？——放到对商业的分析上，然后利用其信息实实在在的创业，一次次的创业。

我决定创业至今不到一年，我已经有点成长了，但是还不至于让我满意。我希望30岁前登上福布斯青年富豪榜——这是突然在我脑中冒出来的，但是我觉得可以朝着这个目标试试。

我知道AI很火，无论是开发、内容创作都正在起飞。但由于我自身是自媒体起家，所以我还是打算先从内容创作出发，进行创业。具体做什么，目标是什么呢？将自媒体不仅做到收支平衡，还要做到超过正常工资，最终是靠它实现成为百万富翁。

这几天怎么了呢？参加了全球游戏IP大会，参加了深科创面试，朋友圈里一堆人正在创业，而且已经很牛逼了——但是我认为自己最好别受到他们进度的影响，专心做自己领域，做到优秀。

最重要的一点，要多写博客记录自己的创业过程。这是一种整理思绪的好方法。

关于状态，还是认可之前自己的想法：要确保生活舒适，对自我的高认可度、自信。创业，其实是和自己进行拉扯，最终是要跟自己和解。

INFP的创业者的毛病，就是容易陷入某种场域、状态之中，忘记主线。

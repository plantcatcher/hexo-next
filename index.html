关于经济学、商业，如何融汇贯通的思考

发表于 2026-03-10 分类于 创业

我把自己目前的学习分为这么几个阶段：

第一阶段：《富爸爸》系列，意识到B象限和I象限才是最正确的选择；

第二阶段：经济学、金融知识积累；

第三阶段：创业课程学习与实践；

这么看来，我的路径似乎很清晰明了，但我当前困惑的点也正是在这儿。首先，简要说明一下路径。在第一阶段，我意识到需要告别打工者思维，从E和S象限，往B和I象限努力。也粗浅了解杠杆在整个金钱游戏中的重要性，以及现金流和资产负债表。第二阶段，我通过阅读基础的经济学书籍，如《小岛经济学》；学习金融知识，如《硅谷101》、《小Lin说》等；以及其它课程，对整个世界的经济运作有了基础的认知。第三阶段，阅读大量硅谷明星公司的传记，如《FaceBook》、《支付战争》、《解密Instagram》、《孵化推特》等，能了解创业者的路程以及给自己打打鸡血；专门学习创业课程，例如《YC创业营》的书籍与斯坦福大学创业课；阅读公司相关书籍，如《一本书读懂财报》、《肖星的财务思维课》，掌握开公司需要知道的内容；看各种商业大佬的访谈和演讲，例如刘强东、段永平、埃隆马斯克、AI圈大佬的播客等，吸取大佬的知识；此外，还有通过参加深圳各种线下活动、讲座、圆桌对谈、峰会、会展，来扩大见识。

好，那么我困惑在哪儿呢？我意识到，虽然知道很多名词，但是并没有很好的将它和别的知识点融汇贯通。加上OpenClaw这个事情让我对科技的速度有点FOMO和焦虑，以及当前入不敷出的状态，所以产生自我怀疑。

但我花了几天时间，到现在我终于想到了一个很好的解决办法：跟AI聊天。我让豆包帮我梳理了整个路径，让它帮我捋清楚知识点。我的极强的行动力、学习能力，加上豆包的提醒、辅助思考、复盘能力，似乎没什么能够阻止我成长的。

首先记住最重要的一句话：

所有商业动作，本质都是：用更低的成本，更快地满足别人的需求，然后把这件事变成可重复、可放大的系统。

我学的所有名词，都是为了这句话服务的。

1 需求/痛点

不是“用户想要什么”这么浅。经济学本质：

·人有欲望，但资源有限；

·他自己解决不了，或解决起来成本太高→这就是痛点。

我做产品，就是帮他们把“成本”降下来。

2 MVP（最小可行产品）

对应小岛经济学里的知识：**先储蓄一点资本（资金、技能等），造个简陋工具试试好不好用，好用再投入更多资本升级。**MVP不是随便做个烂东西，其经济学意义是：

·用最少的成本，验证需求是不是真的，避免把所有资本（时间、资金、能力）砸进一个没人要的东西里。

这就是风险控制，是最基础的经济理性。

3 反馈 → 迭代 → 热循环

这就是市场机制。

·用户给钱 / 点赞 / 关注 = 投票

·没人用 = 你的生产不被市场认可

·改到有人持续用 = 你和市场形成正向循环

生产 → 交换 → 积累资本 → 扩大生产，就是经济增长的最简模型。

4 SOP（标准作业流程）

富爸爸 B 象限的核心就是：SOP。SOP 降低了协作成本、学习成本、犯错成本，让效率指数级上升。

5 商业模式

商业模式 = 你怎么持续赚钱的经济闭环。最朴素的三问：

·你帮谁省成本 / 解决问题？（需求）

·你用什么帮他？（产品 / MVP）

·他为什么愿意持续给你钱？（重复 / 系统 / SOP）

商业模式通了，就是一个小型经济体跑通了。

AI总结的表格

AI的建议

它给我一个最简单的 “贯通训练”：我以后看到任何商业词，只问自己三句：

  1. 它节约了什么成本？
  2. 它提高了什么效率？
  3. 它让哪一环变成可复制？

能回答出来，我就真懂了。

创业准备期的一些思考

发表于 2026-03-08 分类于 创业

OPC（一人公司）在深圳很火，圈子里很多人都在讨论。openclaw目前我还没看到很好的落地场景，而创业圈子里已经有很多在探索，也有很多围观者筹划着进入。焦虑，从众心理，围绕在深圳上空。

我虽然有很多待办事项，但是我依旧焦虑。排除创业初期入不敷出的焦虑，更多的是探索期的迷茫。参加很多创业活动，大致知道了创业的流程。我知道自己还有很多需要学习的，需要培养商业嗅觉、发现需求；制作出解决用户痛点的产品；通过路演、社媒运营等提升曝光量；通过用户反馈来改善产品。此外，还要对产品的未来有长期规划，对公司进行战略布局，学习关于公司的一切（股权、融资，不一而足）。公司方面、个人方面，都有很多需要学习的。

说说我的个人优势吧。

自由职业者，拥有充足的个人时间；

爱好阅读，因此能获取到更加优质的内容；

强大的信息获取能力，我能在短期内收集到各种活动的信息（线上线下）；

极强的学习能力。这点是我在深圳旅居期间认知到的，我能很快理解知识点；

极强的执行力。从决定到执行，一般不超过一周，甚至有些事情，当天决定，当天解决；

心态开放，接受未知，乐意尝试新鲜事物。

劣势暂不在这里说，等我另一种状态下，一种反思、自省的状态下再进行分析。

AGI时代前夕，预测未来的模样？

发表于 2026-03-05 分类于 社会思考

当openclaw活动席卷全球，我再luma上看到世界各地每天都有很多小龙虾的活动，似乎大家都很FOMO。但回顾过去科技的发展，我认为未来一定是”化繁为简“的。所以哪怕现在很多新技术你不明白，倒也不用恐慌。

从DOS系统，到简易的操作界面；从MCP、skills、agent、vibecoding，到现在的小龙虾，我自认为肯定没有到终点。小龙虾的操作何止是复杂，简直是反人类。我感觉没必要FOMO，没必要过度焦虑，因为很大概率你花时间、精力学的技能，过几个月就又迭代了。

当我知道waytoAGI训练营时，是第四期vibe coding进程快结束的时候。但是往前看前面的第一期、第二期、第三期、第四期，我错过了就真损失了什么吗？并没有，我反倒清醒自己保存了精力，保存了时间。站在2026年初，我甚至会认为n8n是什么玩意儿？我不知道这个技能完全不影响。

现在第五期还在规划中，可能下周开课，我报名了，但是我并不是带着FOMO的状态，而是平和的认为可以了解了解。

正确的思维方式是什么？是利用AI辅助我提升效率，让AI为我所用，而不是盲目追求新鲜玩意儿，唯恐落伍。技术肯定是会往简单方向走的，vibe coding只是一个中间态，当开发网站、app不再困难，技术得到平权，似乎可以预见将来APP的没落，又或者是平民化的泛滥？总而言之，在app上赚钱的上个十年的思维模式，似乎不管用了。那么未来究竟会怎么样？我看不清。

欢迎来到我的个人博客

发表于 2026-03-02 更新于 2026-03-08 分类于 社会思考

写在前面

博客，这是个在我看来上个世代的词汇。在我真正接触互联网之后，经历了微博时代、公众号时代，而当下是短视频的时代。我运气很好，在没有看明白这个整体趋势之前，用那山那海那座城的账号在公众号发布地理科普的文章，收获了不少粉丝，并且还实现了盈利。

公众号：那山那海那座城

那是2025年初，我开始陆陆续续写地理科普的文章。其实刚开始我是在小红书（星球小捕手）上发表个人自制地图，经朋友推荐以及评论区有人建议，可以写关于自制地图的地理分析文章，才开始注册了账号。

一开始，确实没有多少浏览量。什么时候爆发的呢？是在我结束长沙三个月的旅居生活，自驾回到老家的过程中，我的智能手表频繁弹出公众号助手的提醒消息，点赞、喜欢、评论…通知都来了。当天浏览量破万。随后便是真正开始了自媒体的生活。

既然开始吃自媒体这碗饭，自然不少关注自媒体圈的内容，也开始阅读到关于一人公司（OPC）、自由职业者（freelancer）、数字游民相关的文章。渐渐的，发现公众号原来已经是一个夕阳产业，除了为自己竟然赶上末班车，靠着公众号养活自己的庆幸之外，更多是对未来的担忧。这时候，我才真正正视这个短视频时代，已然身处其中的时代。

毫无疑问，短视频依旧会是未来几年内主流的呈现方式，大把流量仍会聚集在此。前段时间听罗永浩的播客（罗永浩的十字路口），有句话很值得思考：

海外的视频平台流量高于图文、音频，国内的视频流量占比更高（80%-90%）。

大势所趋，很多人是跟风进入短视频创作平台，而我虽然迟钝，但在这个数据面前已经没有拒绝的理由了。我下一步计划肯定是进入视频领域，做地理科普类视频。

为什么创建博客？

既然短视频仍旧是未来的主流，公众号都没落了，那为什么我还要开启创建一个似乎是上个世纪的博客平台呢？其实理由也是很简单：记录。

短视频平台经过几年的发展，成了当今大伙每天花费时间最长的平台。2026年，随着vibe coding等一众AI功能的起飞，可以预见未来会有更多会被颠覆的东西。前段时间，阅读Sam Alteman的博客，这位open AI的创始人写的文章，让我有种奇怪的错觉：AI时代，个人思想将会是人类最宝贵的东西，而承接人类思想的博客平台将会是人类的自留地。

当然，有很多平台能承接人类的思想，人类思想也有很多种表现形式：文字、图片、声音、视频、装置艺术。播客在海外兴起很多年了，这两年才渐渐在中国起来。我是小宇宙播客的忠实用户，每天必听。那我为什么还是选择了博客，而不是播客呢？同样是个人内容的输出，文字会安静很多。

如何面的未来？

很多人面对未来，特别这个日新月异的时代，会特别恐慌。我身边的大部分人，则是较为麻木、事不关己的态度活着。而我却很兴奋，我会积极拥抱这个多变的时代，我拥抱AI，我学习很多AI的技能，我用AI来开发工具、网站，用AI来制作AI agent智能助理，我甚至开始用AI来创业。这个时代，是一人公司的时代，AI是一人公司最好的助力。一个碳基生命CEO，加上十几个硅基生命员工，能创造一家千万市值的公司。软件时代，提供给年轻人巨大的杠杆，让许多人实现财富自由。当今的AI时代，同样能让我们这一批年轻人，撬动杠杆，实现财富自由与个人价值。

至此，未来已来。

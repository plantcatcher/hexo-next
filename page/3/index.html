解决焦虑的方法之一，是BeExcellentToYourself

发表于 2026-06-02 分类于 创业

社会圈层的人认可你“优秀”，可能还是带着点认知偏差的评判，你自己有很大的操作空间让他人“认为你优秀”。那么什么样的优秀才是真的优秀呢？其实，所谓优秀，就是基于社会规则下的达成某种成就的优秀。就像是通过某个很难的考试，拿到一个很有含金量的奖项，考上一所很牛逼的大学….但认识自己并没有这样的天赋，所以会有社交焦虑，如何解决呢？

昨天和一个小伙伴吃饭，然后去爵士酒吧聊天。聊天多集中在创业相关的话题，然后是闲聊。从晚上8点碰面，到凌晨结束，各自打车离开。谈不上多开心，话题也并没有多有趣，但是这种生活状态让我不禁思考怎么样的人生才算是有趣。当晚算有趣吗？并不算。但也是重新唤起了对赚钱的迫切愿望，就像当年去泰国乘坐豪华游轮时的feeling一样，I feel like I need to earn more money to afford this lifestyle。

然后就是对伴侣的思考。我莫名的很自卑，在此先按下不表吧。我自认为，不应该在这段时间内找对象，让她来跟着我受苦。

很奇怪的感受。在参加很多活动后，我会有想多曝光、多混脸熟的想法。但今晚过后，我反倒开始认为我该沉下心来，去提升自己某方面的能力， 而不是频繁的追逐曝光。我能剖析这种转变，大概率是一种回避、逃避心理，一种自我保护，在没有全副武装、身穿铠甲的情况下，我不愿只身前往“战场”。现在是6月初，我对本月的期望是，潜心修炼。

打零工VS训练营，如何让自己能力最大化？

发表于 2026-05-31 分类于 创业

在微信朋友圈，看到一个打零工的信息，是一个自由职业者的朋友。我浏览了软件一圈，是出卖劳动力的平台，时薪21元左右。让我忽然犹豫——真正缺钱时，这个软件确实能救命。但是在获得外部资金支持的情况下，是否将时间放在AI训练营or产品研发or阅读学习上，更能将自我能力最大化？

站在理性的角度上，无疑后者是最好的选择。不情愿可能是来自于自己这段时间的状态，即意识到你再怎么提升自我，似乎发挥的作用有限（我将其认为是暂时的短时，缺乏前瞻性的表现），如果纠正这点，我会选择重新投入学习的怀抱。

说是创业期，但在我看来就是训练对自我的认知，学习如何掌控自己的生活。那么，如何纠正当前的缺乏远见的认知偏差呢？我能想到的是，重新对其赋予意义。也就是说，认可阅读的作用、训练营的作用、研发产品的作用、锻炼身体的作用、发呆的作用…以及，自我的作用。

创业为什么艰难？商业游戏、金钱游戏本身规则的残酷之外，更重要的一部分是自我的攻略。需要对自己的优劣势有准确的判断，然后将时间和注意力投入到对自我能力的训练上来，然后精进——毫无疑问，任何创业者不可能一开始就是优秀的创业者，总要经过一定的历练。而历练除了想无头苍蝇盲目试错外，还能是有条理、有规划、有针对性训练的自我提升。按照朝着心流的方向走，我需要在定下方案以及赋予意义之后，获取一定的反馈，对行为进行微调。SO，博客平台将承担起记录与提醒的功能。未来我的更多尝试以及心理分析，将会在博客平台一 一被记录。

唯时间不可虚度，唯艺术不可辜负

发表于 2026-05-31 分类于 创业

如题，当我不知道干什么的时候，可以投身于艺术。切不可盲目跟随生活惯性，做出浪费时间的举动。

我需要说这段时间经历了什么吗？可以说。也可以把这段时间脑子里碰撞产生的各种思想一股脑全写出来，毫无逻辑的写出来。

今天是5月的最后一天，时间过的很快，我这段时间经历了不少事情，但总可归为参加活动、维系创业关系。

……

我还是从自我内心开始剖析吧。

我渐渐明白了商业世界的逻辑，人脉是很重要的，但大伙对你的认可只靠混脸熟是不行的，还需要拿出实实在在能证明能力的东西。所以，我需要开发产品，需要学习融资，需要多参加活动，甚至多发表言论。我相信长久混迹于这个创投圈子里，会让我慢慢变得游刃有余，得心应手。但我也深知自己并非热衷呆在这个圈子内，我内心仍旧只喜欢跟自己契合的人交往、合作。

我在某些方面，会吸引某类人接近我，这其实算是我的优势之一，因为这会让我面临更轻松的人际关系。

我还是无法静下心来长时间阅读（短期可以，在通勤时段我还能微信读书一整个小时），我自认为是状态还没调整过来。我深知我的读书角、床尾放着这么多蕴含“真知灼见”的书籍，却无法消受的可叹，可强迫自己阅读其效果及其有限，倒不如——放松、享受艺术、享受音乐。

除了创业，我更应该思考自己的生活本身。我想过什么样的生活？我知道思维锻炼的重要性，也知道假设性问题对锻炼思维的重要性。

我接下来生活做的事情，是要允许思考同时进行的事，允许发散思维的事。阅读是主动引导式思考、听歌欣赏艺术则是发散式思考、玩游戏是天马行空思考、发呆是冥想式思考，都是挺好的选择，唯有刷视频是对思考毫无助长的，所以必须pass.

有一个点我需要注意的，是我需要维系一种较为亲密的能够说心里话的朋友关系。孤独不可取。胡乱说一通，狗屁不通，但是无所谓。

5.摄影and电影，以及参加展览，是我通往艺术与自我内心的自留地；

6.DO NOT 浪费时间玩手机。

好了，我要重新培养习惯，然后坚持下去了。PS：你想进入一个圈子的最好办法，就是多参加圈子内部举办的活动，并且积极认识新朋友，这样慢慢的你将认识圈子内的大部分人。

你能学会任何东西的二八定律

发表于 2026-05-24 分类于 创业

我是相信二八定律的，所以我认为任何领域，投入不算太多的时间，例如几百个上千个小时，你就能掌握这个领域80%的知识。在认可这个前提下，需要注意的就是如何确立目标，认识规则、实践以及不断精进。

那么，我当前需要学会什么呢？学会商业领域大伙认可的能力，具体将这种能力拆开，例如承担责任、诚信等，然后不断践行培养这类能力，即可达到目标。这种方法也可以用到任何领域，需要提前培养的基础能力就是：洞察任何领域人群所认可能力的能力，以及将问题拆开逐个击破的能力、通过正反馈精进能力的能力。所谓任何领域则可以是：语言学习（英语）、拍电影、生活品质、穿搭、艺术鉴赏、谈恋爱。

把人生当成一个地球online游戏，任何领域都是一个副本，掌握每个支线的规则，通过不断试错，最终是要将游戏打通关——唯一要时刻提醒自己的是：人生最宝贵的是时间和注意力。

当环境配不上你的努力、野心以及期待

发表于 2026-05-22 分类于 创业

为什么忽然想聊聊这个话题，是因为这次报名的志愿者活动让我觉得很无趣。

活动名称就不说了，总之是一个AI相关的活动，为期5天，朝九晚六，主要是我每天往返通勤时间是四个小时。当然这只是其中一个原因，还有就是这个环境以及人员的构成，让我觉得之前的努力白费了，辛辛苦苦提升自己，结果混进了挺low的人群中。才是第一天，已经让我开始思考要如何说服自己安稳且不浪费时间的度过这段时期了。

这让我会想起几个月前，报名参加社区图书馆的志愿者，无聊且毫无希望，纯纯大冤种。

我其实早该意识到，我需要一个优秀的环境激励我成长，且我适应能力很强。而糟糕的环境浪费时间不说，还损害我的精力、污染我的精神世界。

我想遇到有意思的人，做有意思的事，小伙伴都在为自己的未来所努力奋斗。我遇到过，所以我相信还会再遇到，这是我当下努力提升自己的动力。而也因此，我并不喜欢和固步自封的人交朋友，也不喜欢和眼界不高的人交朋友，更不喜欢和boring的人交朋友。SO，很遗憾，我过去结识的大部分人PASS掉了。我并不认为自己是慕强，因为我对社会公认牛逼的那类人并不是很感兴趣，我更喜欢有趣的人，以及有思想的人，或者对未来有憧憬的人。我之所以这么说，是因为我真的遇到了这些好朋友，跟TA们聊天很开心，相处很愉快。

回到标题，当环境配不上你的努力、野心以及期待，我该怎么办？当然是坚定不移的提升自我，不固步自封，争取早日“脱离苦海”了。

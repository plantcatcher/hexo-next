谈谈女人缘与提升自我

发表于 2026-05-08 分类于 创业

我秉承着这么一个交友观：朋友不限定男女。所以我会去交聊得来的小伙伴，而不是根据性别来进行评判。因此，在深圳交到了不少优秀的小伙伴朋友，有来自TEDxShenzhen的，也有来自于黑客松比赛以及一些其它活动的。

但我能看到一个较为明显的现象，参加活动期间、以及活动结束后，异性小伙伴会很乐意接着在微信上跟我瞎聊天。我知道自己不是那种建模脸，只能推测她们之所以乐意跟我聊天，或者约线下逛街、吃饭，是因为我是一个很好的倾听者。

还有一些我环球旅居期间认识的小伙伴，也会在有困惑时找我聊天，有些是选择微信语音通话，有些是选择一问一答。给我的一致反馈就是，很高兴我解决了她们的问题，至于什么问题呢？我觉得自己提供的仅仅是情绪价值。这种算是天赋还是后天养成的呢？我倾向于后天养成说。

在外人看来，我是一个闷声干大事的人。每过一段时间，我总能做出一些让大伙惊讶、佩服的事情。而我知道自己其实只是追随内心的想法，一步步提升自我——我甚至不知道未来究竟如何。其实就是走一步看一步，随之而来的是养成了一种奇怪的性格：对很多事情都是无所谓的状态，得失心没这么重了。SBTI测试，我的结果是OJBK，很莫名其妙，但想想也正常，经历过几次大的冲击之后，我确实看开了很多。

黑客松之后，确实有戒断反应，而且我对自己此前的计划开始进行推倒重建。但重建的这个过程很漫长，我把这段时间称之为“沉沦期”，每天不知道干什么，对社会似乎没产生贡献，自我认同低。但也恰恰是这段时间，身边多了几个聊得来的小伙伴，能约去香港体验apple vision pro，能约去线下吃饭，能在微信聊对方的童年记忆，能聊BCI科技的预测——焦虑，但是还算幸运。

在上一篇博客文章中我有提过，我需要沉下来提升自己。在经过这两天的自我暗示，我开始慢慢丢掉脑子里那些杂乱的念头，能专注于思考当下以及更远的事情。主要是心态回归平和，能正常进行阅读、学习、实践，我的脑子在飞速进化。我当然想早点上牌桌，但是我也知道能力不足，所以先好好沉寂一段时间吧。

体验VisonPro后，谈谈未来的世界

发表于 2026-05-06 分类于 创业

昨天，跟黑客松的小伙伴一块去香港体验了Apple Vison Pro，在此想谈谈对未来世界的看法。

首先，AVP作为一款2023年出来的产品，这几年其实并没有什么新的升级，无论是在硬件还是软件层面。手机的更新换代不用提，哪怕是电动车的升级速度也远超于AR、VR、XR行业。所以我对当下的判断是，这个行业仍在探索期。

AVP里的三维呈现方式有三种：一种的信息本身是二维，放到一个虚拟的三维空间中；一种是通过算法把信息转换为三维；一种是带交互的三维互动软件（当下教育类居多）。是否值得投入精力去开发三维应用？或许可以试试，但我仍在犹豫。

这次一块去香港的小伙伴，曾经参加过创业营，有不少实战经验。她对乔布斯、苹果公司是很认可的，且对马斯克也是很佩服的。我的想法跟她差不多。沟通下来，有很多点是相同的，很开心能朋友这些志同道合的小伙伴。

深圳的一个优势，是不缺优秀的人，和大量优质的活动。因此，我在深圳挺好。

即将去香港体验科技，说说近况

发表于 2026-05-04 分类于 创业

说起近况，只能说依旧目标迷失。参加黑客松志愿者后，后劲有点大，出现了戒断反应。我对当前生活进行了反思，算是破了些旧行为，但是新的行为准则还没有建立起来，所以我这段时间一直是无头苍蝇的状态。

幸好黑客松期间认识了一些优秀的小伙伴，约了一个小伙伴明天一块去香港体验Apple Vision Pro, 期待一波。我的生活呢，首先一直在支出，却没有收入，所以很是焦虑。而且已经有点对创业感到模糊，对商业失去了信心，有种投入后发现收效甚微的感觉。所以，可以说当前的我还在调整状态中。

我并没有持续产出，输入也产生了倦怠，所以更多时候我是在漫无目的地刷手机，不想玩游戏，只是单纯且无聊地看乏味的B站。我觉得这段时间是很值得记录下来的，但是记录什么呢？记录感受吧。

我知道自己有一些长处，但是对我而言这些长处并没有带给我自信，而是对某些场面比较无畏罢了——因为我知道哪怕靠随机应变，我也能应付那些场面。但下一步？我的短处在哪？我需要提升哪方面的能力？毫无头绪。还有，我黑客松志愿者期间的表现，究竟是我在深圳前段时间努力提升自己导致的，还是我本人就是如此，我也不清楚，这就让我无法客观的评价在深圳创业前期的准备正确与否。不可否认，在深圳这段时间，我的人脉有了很大的提升，但我个人能力、眼界、修养是否有长进，需要打个问号。

玩商业游戏，你会发现机会会随着你按部就班做人物出现，并让你升级到下一等级。而现实生活中，光想不做则什么都不会改变，甚至于哪怕你做了些努力，其收效甚微无法影响生活。我虽然在早期，开发了些网站、做了博客，但是对我究竟有什么帮助呢？不好说。那我究竟做什么才是真正有帮助的？——在这里单指输出、行动方面的。（输入部分我知道什么是对的，但无奈没啥坚持的欲望，例如学习英语、学习商业知识、研究市场，作为行动则是阅读、听讲座播客、做线上调研）。

金钱游戏果然不是这么容易玩的。

我为什么一直在创业和工作之间犹豫？因为创业，我目前并没有探索出商业闭环，而去工作，只是完成别人已经实现商业闭环的某个环节，公司肯招我，表示需要我某方面的能力，因此对我并不会有太大的难度。

我已经很不要脸的投了字节跳动的简历，虽然我知道大厂很难进，但是试试并不会有什么损失，所以大胆投递。此外，在参加活动以及这段时间无所事事中，我有点想念稳定被托住的感觉，我需要一个稳定的社交环境来找到自己在社会中的定位。所以，我可能还是会选择投简历到各个公司，公司类型包括：地理类、机器人类、一般销售类。

BOSS直聘上投简历，总是容易石沉大海，所以我更倾向于发电子邮件。

为了让自己的简历更丰富多彩，我决定丰富自己的地理网站，证明有能力开发webgis的能力，以及各种关于数据库、服务器、前端框架等内容的了解。如此，接下来的日子就有努力的方向了——地理网站的开发。

我的私心是，地理网站开发，一方面能丰富我的简历，一方面能测试市场的反应。每天花费几个小时开发，能让我的生活不至于太虚度，还能满足前面的需求，其实是一举多得的。当然，这只是我当下的想法，未来如何，还是未知。

AI时代如何把握时代脉搏？

发表于 2026-04-29 分类于 创业

AI时代，只要你学得够慢，你就不用学。因为各个大厂、AI本身会不断进步，我们不需要掌握在这个巨变中的一个小变量，我们需要的是掌握大方向，看懂真正的未来。

所以，我认为去掌握宏观方向才是主要的。放弃成为工厂的螺丝钉，去成为统筹全局的产品经理，才能把握住AI时代的脉搏。因此，我的学习方向，应该从小领域转为宏观研究（但经济学，仍需要进行微观研究）。——谁能更准确的预言未来，并去践行，才有机会脱颖而出。

在这个前提下，我需要阅读的书籍是类似于凯文·凯利《必然》这种，需要听的访谈是Sam Altman、Elon Musk等大佬的。此外，我还需要输出，因此目前计划在小红书上更新自己阅读、观看访谈等内容后的感悟、总结。

焦虑又毫无目标的一天

钱，这个问题一直围绕着我，我的想法也围绕着是否坚持思考B象限的内容or重新找一份工作老老实实上班。我知道目前的资金还够自己撑一个月，但是我焦虑的点还是——虽然看了这么多创业的书，但不得其法，不知道如何发掘痛点、觉察需求。我也知道无论如何，该坚持的行为、习惯还是要坚持，但焦虑一直围绕着我，自我怀疑，无可奈何。

阅读无法投入，仅能每天一章节缓慢推进；创作毫无动力，已经十几天没有动作了；生活也是一团糟，感觉自己在荒废时间；能力毫无长进。

是因为长时间没有创作、不进行输出，所以我才焦虑的吗？是因为没有行动，所以我才焦虑的吗？不清楚，但我可以尝试改变，重新回归创作日常吧。

重新创作，重新学习新内容。在看硅谷101的内容，是关于airbnb的创始人访谈，边看边思考。渐渐的，我忽然意识到为什么我这段时间状态很差——因为我线下见到很多有能力的人，对比下来我能力很差，所以我自惭形秽了。这似乎能解释为什么是这段时间，我开始自我怀疑。但我知道，为了应付未来更大的挑战，现在是我为数不多可以积累知识、能力的空闲时期，所以我要重新安排自己的日程。

目前本周计划：

1.不刷B站，仅看已经下载好的商业访谈内容；

2.接着阅读商业书籍、传记；

3.坚持创作公众号内容，继续尝试地理科普视频创作；

4.坚持英语学习，刷PTE、阅读英语文章、做笔记；

对的，确实是有可能因为很长时间没有接触过厉害的人，所以下意识自惭形秽。我需要改变思想，从自卑变为积极向他们学习，一点一点模仿他们并给自己认可的反馈。

这也让我意识到，我的博客内容，应该减少抱怨，多往积极方面去思考，寻找解决问题的办法，而不是困在问题之中郁郁寡欢。

《MilionDollarWeekend》笔记

发表于 2026-04-19 分类于 创业

这是我拼多多买的一本英文书（中文版的太贵了，英文版只要10块），当天到，当天开始看。目前已经发现这本书干货内容很多，书不从商业知识、技巧出发来赚到百万美元，而是通过改变一个人自身（包括思维、行动、心态、性格等）。

我刚看到开头，书中说很多创业者之所以失败其实就是两个原因——两个fear:

  1. Fear to starting
  2. Fear of asking

这两点直接戳中的我的weekness，这正是我需要做出改变的。针对第一条，我不需要做足准备再开始，不需要找朋友来克服自己内心的insecurities，做最小尝试。第二条，则是需要多问、多去表达自己的需求。——我不希望自己对这本书依旧是看过就算，不求甚解。我需要借着这本书来改变自己的思维。

我看了不少商业书籍，为什么就是做不到成功创业？而有些人根本没看过书，却能初创公司就风生水起？我意识到，是性格、思维、心态。我之前学的都是“外挂”，是财商、经济学、产品、营销、商业模式……这些都是工具、方法、认知。

没有这些，创业很容易踩坑；但光有这些，我依然动不起来。

真正卡住我的，是“内部系统”。即怕被拒绝、怕丢脸、想太多做太少、容易内耗、不敢开口要结果、不敢定价不敢收钱、总觉得自己还没准备好。我渐渐明白——赚大钱的人，认知不一定比我深，但行动力和脸皮一定比我厚。

跟朋友吃串串香的一天周六

发表于 2026-04-18 分类于 创业

在实现了一个小目标之后，我陷入了很长一段时间的不作为。我似乎丧失了新的动机，去学习、去奋斗、去成长。一觉睡到下午一点，朋友打电话过来，问我想不想一块吃个饭，我说好。下午两点，他来到了我在深圳的出租屋。然后在还没确定去哪个地方的时候，搭上了前往坪地六联的3号地铁。

在下错站后，我们又多走一个站，到附近的串串香吃饭。期间聊他的新工作，聊我对未来的想法和打算。吃完后，本来想去别的地方逛逛，但是朋友昨晚没睡好，所以让他先回家，我也回出租屋接着琢磨自己的未来。

现在是什么个情况呢？就是什么也不想干，刷手机短视频也不想。但还好我提前想好后路——当我丧失动力时，坚持完成每天的计划：

  1. 阅读看不下去的书一个章节；
  2. 刷PTE题目；
  3. 做20个俯卧撑；

因为我还是相信杠杆的力量，即我每天不易察觉的进步，能对我的未来产生很大的影响。

我感觉自己的情绪是呈现周期性变化的，现在恰好处在低迷期。应对低迷期的自己，除了坚持每天做对的事之外，我还应该想一些别的方法。但不得不承认现在我的做法很差劲：我下载有意思的综艺《新世界：虚拟货币战争》，玩各种商业游戏。虽然这两个方法都是和商业相关，但是我不太确定能对我产生多大的帮助。

另外，广西一个女性朋友最近去北海旅游，她在群里发的照片也是让我心烦意乱。我想放松，但是我不能，因为我要尽快“三十而立”。
